# 角色卡系统设计

## 一、设计理念

### 1.1 核心定位

**角色卡是平台的"IP单元"——可跨世界使用的角色身份，便于分发、交易和破圈。**

一句话理解："创建一个角色，TA可以进入任何世界冒险。"

### 1.2 设计原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **一键创建** | 上传图片或描述，AI生成完整角色 | 图片/文字→角色卡 |
| **跨世界通用** | 角色可以在不同世界间旅行 | 标准化属性系统 |
| **成长可视化** | 角色成长清晰可见 | 属性面板+成就展示 |
| **社交货币** | 稀有角色成为身份象征 | 稀有度+外观定制 |
| **易于分发** | 角色卡本身就是分享内容 | 卡片式展示+社交分享 |

```

- 外观图片
- 属性分配（战士型）
- 技能组合（近战攻击）
- 背景故事
```

#### 方式二：图片上传

```
用户上传：角色概念图

AI识别：
- 外观特征提取
- 风格分析
- 自动生成角色卡
```

```
用户选择：战士模板

系统生成：
- 预设属性
- 预设技能
- 可自定义外观
```

### 5.2 交易机制

```python
class CharacterCardTrading:
    def list_for_sale(self, character_id, price):
        """上架角色卡"""
        character = self.get_character(character_id)
        character.status = "for_sale"
        character.price = price
        self.market.add_listing(character)

    def buy_character(self, buyer_id, listing_id):
        """购买角色卡"""
        listing = self.market.get_listing(listing_id)
        buyer = self.get_user(buyer_id)

        if buyer.balance >= listing.price:
            # 扣除买家资金
            buyer.deduct(listing.price)

            # 给卖家资金（扣除平台手续费）
            seller = listing.seller
            seller.add(listing.price * 0.7)

            # 转移所有权
            listing.character.owner = buyer_id
            listing.character.status = "owned"

            return True
        return False
```

### 6.2 属性转换

```python
class AttributeConverter:
    def convert(self, character, target_world):
        """转换角色属性到目标世界"""
        rules = target_world.attribute_rules
        converted = {}

        for attr, value in character.attributes.items():
            if attr in rules.mapping:
                converted[rules.mapping[attr]] = value
            else:
                converted[attr] = value

        # 应用世界修正
        for modifier in rules.modifiers:
            converted[modifier.attr] *= modifier.factor

        return converted
```

### 7.2 组队系统

```python
class PartySystem:
    def create_party(self, leader_id):
        """创建队伍"""
        party = Party(leader=leader_id)
        party.members = [leader_id]
        party.max_size = 4
        return party

    def invite_member(self, party_id, member_id):
        """邀请成员"""
        party = self.get_party(party_id)
        if len(party.members) < party.max_size:
            party.pending_invites.append(member_id)
            return True
        return False

    def enter_world(self, party_id, world_id):
        """队伍进入世界"""
        party = self.get_party(party_id)
        world = self.get_world(world_id)

        # 检查所有成员是否满足进入条件
        for member_id in party.members:
            member = self.get_character(member_id)
            if not world.meets_requirements(member):
                return False

        # 同步进入世界
        for member_id in party.members:
            self.teleport(member_id, world_id)

        return True
```
