# 世界书系统设计

## 一、设计理念

### 1.1 核心定位

**世界书是平台的"内容容器"——定义一个世界的全部规则，让AI自动生成画面和剧情路径。**

一句话理解："写一段话，AI就帮你生成一个世界。"

### 1.2 设计原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **极简创建** | 用户一句话就能创建世界 | 自然语言→结构化数据 |
| **深度可编辑** | 高级用户可以精细调整 | 可视化编辑器 |
| **模板继承** | 基于模板快速创建 | 模板市场 |
| **版本管理** | 世界可以迭代更新 | Git式版本控制 |
| **可组合** | 世界可以引用其他世界元素 | 模块化设计 |

### 2.2 一句话创建

```python
class WorldBookGenerator:
    """一句话生成世界书"""

    def create_from_prompt(self, user_prompt: str) -> WorldBook:
        """
        用户输入："我想要一个赛博朋克风格的城市，巨型公司统治着一切"

        AI处理流程：
        1. 解析用户意图
        2. 生成完整世界书结构
        3. 填充默认值
        4. 返回可编辑的世界书
        """

        # 使用LLM解析
        parsed = self.llm.parse_intent(user_prompt)

        # 生成基础结构
        world_book = WorldBook()
        world_book.meta.name = parsed.name or "未命名世界"
        world_book.rules.one_liner = user_prompt
        world_book.rules.genre = parsed.genre
        world_book.rules.era = parsed.era

        # 自动生成地理
        world_book.geography = self.generate_geography(parsed)

        # 自动生成势力
        world_book.factions = self.generate_factions(parsed)

        # 自动生成剧情主线
        world_book.story = self.generate_story(parsed)

        # 自动生成视频参数
        world_book.video_params = self.generate_video_params(parsed)

        return world_book
```

class PreGenerationService:
    """预生成热门内容"""

    def pre_generate_popular_worlds(self):
        # 获取热门世界
        popular = self.get_top_worlds(limit=100)

        for world in popular:
            # 预生成所有节点的视频
