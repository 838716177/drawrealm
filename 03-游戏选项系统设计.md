# 游戏选项系统设计

## 一、设计理念

### 1.1 核心定位

**游戏选项是平台的"交互核心"——将复杂游戏简化为选择，降低门槛，提升叙事深度。**

一句话理解："看视频，做选择，你的决定改变世界。"

### 1.2 设计原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **极简交互** | 每个节点最多3-4个选项 | 限制选项数量 |
| **有意义的选择** | 每个选项都有后果 | 蝴蝶效应系统 |
| **即时反馈** | 选择后立即看到结果 | 预生成内容 |
| **视觉化** | 选项本身就有视觉表现 | 动态UI |
| **可预测性** | 玩家能预判选项后果 | 提示系统 |

## 四、蝴蝶效应系统

### 4.1 标志系统

```python
class ButterflyEffect:
    """蝴蝶效应系统"""

    def __init__(self):
        self.flags = {}  # 全局标志
        self.relationships = {}  # 关系值
        self.world_state = {}  # 世界状态

    def set_flag(self, flag_name, value):
        """设置标志"""
        self.flags[flag_name] = value

    def check_flag(self, flag_name):
        """检查标志"""
        return self.flags.get(flag_name, False)

    def modify_relationship(self, character_id, value):
        """修改关系值"""
        if character_id not in self.relationships:
            self.relationships[character_id] = 0
        self.relationships[character_id] += value

    def get_consequences(self, flag_name):
        """获取标志的后续影响"""
        consequences = []

        # 检查所有节点，看哪些受这个标志影响
        for node in self.all_nodes:
            if node.requires_flag(flag_name):
                consequences.append(node)

        return consequences
```

### 5.2 选项注入时机

```python
class VideoOptionInjector:
    """选项注入系统"""

    def __init__(self):
        self.video_player = VideoPlayer()
        self.option_display = OptionDisplay()

    def play_node(self, node):
        """播放节点"""
        # 1. 播放视频
        self.video_player.play(node.video_url)

        # 2. 在指定时间点显示选项
        if node.option_timing:
            self.video_player.at_time(
                node.option_timing,
                lambda: self.show_options(node.options)
            )
        else:
            # 默认在视频结束时显示选项
            self.video_player.on_end(
                lambda: self.show_options(node.options)
            )

    def show_options(self, options):
        """显示选项"""
        # 过滤不可见选项
        visible_options = [opt for opt in options if opt.is_visible()]

        # 显示选项UI
        self.option_display.show(visible_options)

        # 启动计时器（如果有时间限制）
        if options[0].time_limit:
            self.start_timer(options[0].time_limit)

    def on_option_selected(self, option):
        """选项被选择"""
        # 1. 应用选项效果
        self.apply_effects(option.effects)

        # 2. 检查成功/失败
        if option.success_rate:
            if random.random() < option.success_rate / 100:
                self.goto_node(option.success.next_node)
            else:
                self.goto_node(option.failure.next_node)
        else:
            self.goto_node(option.next_node)
```

### 6.2 战斗结果

```python
class CombatSystem:
    def resolve_combat(self, player_choice, enemy):
        """解析战斗"""
        # 1. 计算玩家伤害
        player_damage = self.calculate_damage(
            attacker=player,
            defender=enemy,
            action=player_choice
        )

        # 2. 应用伤害
        enemy.hp -= player_damage

        # 3. 检查敌人是否倒下
        if enemy.hp <= 0:
            return CombatResult.VICTORY

        # 4. 敌人反击
        enemy_damage = self.calculate_damage(
            attacker=enemy,
            defender=player,
            action=enemy.ai_choose_action()
        )

        player.hp -= enemy_damage

        # 5. 检查玩家是否倒下
        if player.hp <= 0:
            return CombatResult.DEFEAT

        return CombatResult.CONTINUE
```

## 七、存档系统

### 7.1 自动存档

```python
class SaveSystem:
    def auto_save(self, player_state):
        """自动存档"""
        save_data = {
            "timestamp": datetime.now(),
            "player": player_state,
            "world_flags": self.butterfly_effect.flags,
            "current_node": player_state.current_node,
            "history": player_state.choice_history
        }

        # 保存到云端
        self.cloud_save.save(save_data)

        # 本地备份
        self.local_save.save(save_data)

    def load_save(self, save_id):
        """读取存档"""
        save_data = self.cloud_save.load(save_id)

        # 恢复玩家状态
        player_state = save_data["player"]

        # 恢复世界标志
        self.butterfly_effect.flags = save_data["world_flags"]

        # 恢复到当前节点
        self.goto_node(save_data["current_node"])

        return player_state
```

## 九、总结

### 9.1 核心设计

```
游戏选项 = 极简交互 + 有意义的选择 + 即时反馈 + 视觉化
```

### 9.2 用户旅程

```
观看视频
    ↓
出现选项
    ↓
做出选择
    ↓
看到结果
    ↓

```

### 9.3 关键指标

| 指标 | 目标 |
|------|------|
| 选项响应时间 | < 1秒 |
| 视频加载时间 | < 3秒 |
| 选项理解时间 | < 5秒 |
