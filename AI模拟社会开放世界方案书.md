# 绘境Online - AI原生互动娱乐平台方案书

## 一、重新理解：Elseland的正确解法

### 1.1 核心公式

```
角色卡 + 世界书 + AI视频生成 = 新互动娱乐平台
```

**不是**：模拟社会 → 社会经济体
**而是**：互动视频游戏平台 → 内容消费 + 社交体验

### 1.2 技术路径：AI视频 = 3A游戏视觉效果

```
image2（图片生成）
     ↓ 稳定画面帧
Seedance 2.0（视频生成）
     ↓
视频内容 = 游戏画面
     ↓
选项节点 → 剧情分支
```

**核心优势**：
- 绕开游戏制作的复杂性
- 用选项代替复杂操作
- 打BOSS只有战胜/失败两种结局
- 但角色属性、物品可以成长

### 1.3 内容三角

```
        世界书
      /   |   \
     /    |    \
    /     |     \
   /      |      \
  ▼       ▼       ▼
世界规则 画面生成  游戏路径
  │       │        │
  └───────┴────────┘
           │
           ▼
       AI视频内容
           │
           ▼
       玩家互动选项
```

## 三、核心模块详解

### 3.1 世界书（World Book）

世界书是内容的核心，定义一个世界的全部规则：

```yaml
world_book:
  name: "赛博朋克都市"
  genre: "未来/科幻"

  # 世界规则
  rules:
    technology_level: "high"        # 科技水平
    magic_level: 0                 # 魔法水平
    social_structure: "corporate"  # 社会结构
    conflict_type: "corporate_war" # 冲突类型

  # 地理环境
  geography:
    main_city: "新上海"
    districts:
      - name: "核心区"
        description: "巨型公司总部，高楼林立"
        risk_level: "low"
      - name: "工业区"
        description: "工厂与贫民窟并存"
        risk_level: "medium"
      - name: "废墟区"
        description: "废弃街区，危险横行"
        risk_level: "high"

  # 势力分布
  factions:
    - name: "星际集团"
      type: "corporation"
      alignment: "neutral"
      influence: 80
    - name: "反抗军"
      type: "resistance"
      alignment: "chaotic"
      influence: 20

  # 剧情节点
  story_nodes:
    - id: "start"
      title: "降落在新上海"
      description: "你从穿梭机走下，霓虹灯照亮了天空"
      options:
        - text: "前往核心区"
          next_node: "core_arrival"
          requirements: null
        - text: "前往工业区"
          next_node: "industrial_arrival"
          requirements: null

    - id: "core_arrival"
      title: "核心区"
      description: "高楼林立，全息广告无处不在"
      enemy: "security_bot"
      options:
        - text: "潜入公司"
          next_node: "corporate_heist"
          skill_check: "stealth"
        - text: "正面交涉"
          next_node: "corporate_talk"
          skill_check: "charisma"

  # 视频生成参数
  video_params:
    style: "cyberpunk"
    color_palette: ["neon_blue", "neon_pink", "dark_purple"]
    visual_elements: ["hologram", "flying_cars", "megastructure"]
```

```python
    async def generate_episode(self, world_book, node, character):
        # 1. 准备场景描述
        scene_desc = self.build_scene_description(world_book, node, character)

        # 2. 生成关键帧
        keyframes = await self.image_gen.generate(
            prompt=scene_desc,
            style=world_book.video_params.style,
            count=6  # 前后各3张
        )

        # 3. 生成视频片段
        video = await self.video_gen.generate(
            keyframes=keyframes,
            description=node.description,
            duration=5  # 5秒
        )

        # 4. 添加选项
        final = self.add_option_overlay(video, node.options)

        return final

    def build_scene_description(self, world_book, node, character):
        # 组合世界设定 + 节点描述 + 角色外观
        return f"{world_book.genre} scene: {node.description}, character: {character.name}"
```
