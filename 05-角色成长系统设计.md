# 角色成长系统设计

## 一、设计理念

### 1.1 核心定位

**角色成长是平台的"留存核心"——让玩家投入时间获得可见的进步，形成沉没成本。**

一句话理解："你的角色在成长，你舍不得离开。"

### 1.2 设计原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **可见成长** | 每次游玩都有进步感 | 即时反馈+数据可视化 |
| **多元路径** | 不同玩家有不同成长方向 | 多分支成长树 |
| **社交展示** | 成长成果可以炫耀 | 排行榜+成就系统 |
| **长期目标** | 有值得追求的终极目标 | 稀有奖励+称号 |
| **适度随机** | 惊喜感驱动持续投入 | 掉落随机+抽卡 |

### 2.2 等级曲线

```python
class LevelSystem:
    def __init__(self):
        self.base_exp = 100
        self.growth_rate = 1.2
        self.max_level = 100

    def exp_for_level(self, level):
        """计算升到某级所需经验"""
        if level <= 1:
            return 0
        return int(self.base_exp * (self.growth_rate ** (level - 1)))

    def total_exp_for_level(self, level):
        """计算从1级到某级总共需要多少经验"""
        total = 0
        for i in range(1, level):
            total += self.exp_for_level(i)
        return total

    def level_from_exp(self, exp):
        """根据经验计算等级"""
        level = 1
        while exp >= self.exp_for_level(level + 1):
            exp -= self.exp_for_level(level + 1)
            level += 1
            if level >= self.max_level:
                break
        return level
```

```python
class AttributeGrowth:
    def __init__(self, character_class):
        self.character_class = character_class
        self.growth_patterns = {
            "warrior": {
                "strength": 3.0,
                "agility": 1.5,
                "intelligence": 0.5,
                "vitality": 2.5,
                "charisma": 1.0,
                "luck": 1.0
            },
            "mage": {
                "strength": 0.5,
                "agility": 1.0,
                "intelligence": 3.0,
                "vitality": 1.5,
                "charisma": 1.5,
                "luck": 1.5
            },
            "assassin": {
                "strength": 1.5,
                "agility": 3.0,
                "intelligence": 1.0,
                "vitality": 1.0,
                "charisma": 1.0,
                "luck": 2.5
            }
        }

    def calculate_growth(self, level_up):
        """计算升级时的属性增长"""
        pattern = self.growth_patterns[self.character_class]
        growth = {}

        for attr, rate in pattern.items():
            # 基础增长
            base = random.randint(1, 3)
            # 职业加成
            bonus = rate * random.uniform(0.8, 1.2)
            growth[attr] = int(base * bonus)

        return growth
```

### 4.3 技能升级

```python
class SkillSystem:
    def upgrade_skill(self, character, skill_id):
        """升级技能"""
        skill = character.get_skill(skill_id)

        if skill.level >= skill.max_level:
            return False, "技能已满级"

        # 检查前置条件
        if skill.required_level > character.level:
            return False, "等级不足"

        if skill.required_skill_points > character.skill_points:
            return False, "技能点不足"

        # 升级技能
        skill.level += 1
        character.skill_points -= skill.required_skill_points

        # 提升效果
        skill.power = skill.base_power * (1.1 ** skill.level)
        skill.duration = skill.base_duration * (1.05 ** skill.level)

        return True, "升级成功"

    def learn_skill(self, character, skill_id):
        """学习新技能"""
        skill_template = self.db.get_skill_template(skill_id)

        # 检查职业匹配
        if character.class not in skill_template.available_classes:
            return False, "职业不匹配"

        # 检查前置技能
        for prereq in skill_template.prerequisites:
            if not character.has_skill(prereq.skill_id, prereq.level):
                return False, f"需要前置技能：{prereq.name}"

        # 学习技能
        new_skill = Skill(
            template_id=skill_id,
            level=1,
            exp=0
        )
        character.skills.append(new_skill)

        return True, "学习成功"
```

### 5.2 装备属性

```python
class Equipment:
    def __init__(self, template_id):
        self.template = self.db.get_equipment_template(template_id)
        self.level = 1
        self.quality = self.roll_quality()
        self.stats = self.calculate_stats()
        self.enchantments = []

    def roll_quality(self):
        """随机品质"""
        roll = random.random()
        if roll < 0.01:
            return "legendary"
        elif roll < 0.05:
            return "epic"
        elif roll < 0.15:
            return "rare"
        elif roll < 0.40:
            return "uncommon"
        else:
            return "common"

    def calculate_stats(self):
        """计算装备属性"""
        base_stats = self.template.base_stats
        quality_multiplier = {
            "common": 1.0,
            "uncommon": 1.2,
            "rare": 1.5,
            "epic": 2.0,
            "legendary": 3.0
        }

        stats = {}
        for stat, value in base_stats.items():
            stats[stat] = int(value * quality_multiplier[self.quality])

        return stats

    def upgrade(self):
        """强化装备"""
        if self.level >= 10:
            return False, "已达最大强化等级"

        self.level += 1
        for stat in self.stats:
            self.stats[stat] = int(self.stats[stat] * 1.1)

        return True, "强化成功"
```

### 6.2 称号系统

```python
class TitleSystem:
    def __init__(self):
        self.titles = {}

    def unlock_title(self, character, title_id):
        """解锁称号"""
        title = self.db.get_title(title_id)
        character.titles.append(title)

        # 发送通知
        self.notify(character, f"解锁新称号：{title.name}")

    def equip_title(self, character, title_id):
        """装备称号"""
        if title_id not in [t.id for t in character.titles]:
            return False, "未解锁该称号"

        character.equipped_title = title_id
        return True, "装备成功"

    def get_title_effects(self, character):
        """获取称号效果"""
        if not character.equipped_title:
            return {}

        title = self.db.get_title(character.equipped_title)
        return title.effects
```

## 十、总结

### 10.1 核心设计

```
角色成长 = 可见进步 + 多元路径 + 社交展示 + 长期目标 + 适度随机
```

### 10.2 用户旅程

```
开始游戏
    ↓
获得经验
    ↓
升级角色
    ↓
解锁技能/外观
    ↓
挑战更强内容
    ↓
获得稀有奖励
    ↓
展示成就
    ↓
持续投入
```

### 10.3 关键指标

| 指标 | 目标 |
|------|------|
| 升级频率 | 每2-3小时1级 |
